## 📋 第一次使用：新組別註冊
首次登入前，每組需要先完成一次註冊。
1. 開啟上方連結
2. 點選 **「📝 新組別註冊」**
3. 依序填入：
   - **學期班級**：從下拉選單選擇（如 `1142甲班`）
   - **組別**：從下拉選單選擇（如 `第3組`）
   - **設定本組密碼**：自訂一組密碼，**請全組記住，之後每次登入都需要**
   - **再次輸入確認**
4. 點 **「完成註冊並登入」**
> ⚠️ 每組只需要註冊一次，之後直接登入即可。
---
## 🔑 每回合登入與提交決策
1. 開啟同學入口網址
2. 選擇 **學期班級** 與 **組別**，輸入本組密碼，點 **「登入」**
3. 登入後進入決策畫面，調整三項資產配比：
| 資產 | 說明 |
|------|------|
| 📈 股票 | 高風險、高報酬 |
| 📋 債券 | 低風險、穩定收益 |
| ₿ 加密貨幣 | 極高風險、極高波動 |
4. 三項合計須為 **100%**
5. 調整 **⚡ 槓桿倍數**（放大獲利與虧損）
6. 確認後點 **「💾 儲存本回合決策」**
> ✅ 出現「決策已儲存！」即代表成功提交，教師端會同步收到。
---
## ⚠️ 注意事項
- 每回合截止前可以**重複修改**，以最後一次儲存為準
- 若老師宣布「回合鎖定」後，畫面會顯示紅色提示，此時**無法再修改**
- 換裝置登入時，請用**相同的班級、組別、密碼**，系統會自動同步記錄
- 忘記密碼請聯繫同組同學，或告知老師由教師端協助處理
